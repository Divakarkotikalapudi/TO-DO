# Advanced To-Do App

A modern and responsive To-Do List application built using HTML, CSS, and JavaScript.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Filter Tasks
  - All
  - Active
  - Completed
- Task Statistics
- Responsive Design
- Modern UI
- Animations
- Local Storage Support

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Concepts Used

- DOM Manipulation
- Event Handling
- CRUD Operations
- localStorage
- Array Methods
- Responsive Web Design
